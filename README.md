# Instruções para Download da Base de Dados

O arquivo CSV **não está incluído** neste repositório por limitações de tamanho e por respeito aos termos de uso do Kaggle.

## Como Baixar

1. Acesse: [https://www.kaggle.com/datasets/datahackers/state-of-data-brazil-20242025](https://www.kaggle.com/datasets/datahackers/state-of-data-brazil-20242025)
2. Faça login ou crie uma conta gratuita no Kaggle
3. Clique em **"Download"**
4. Salve o arquivo CSV nesta pasta (`dados/`) com o nome: `state_of_data_2024.csv`

## Sobre o Dataset

- **Fonte:** Data Hackers em parceria com Bain & Company
- **Respondentes:** 5.217 profissionais de dados do Brasil
- **Colunas:** 400+
- **Período:** 2024–2025
- **Dicionário de dados:** disponível na página do dataset no Kaggle (aba "Data")

## Estrutura Esperada Após Download

```
dados/
├── README.md                  ← este arquivo
└── state_of_data_2024.csv     ← arquivo a ser baixado manualmente
```
